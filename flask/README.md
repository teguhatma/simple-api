# Simple API for Flask

Open browser and go to address http://127.0.0.1:5000/api/hello-world \
This API will return: \
`{ "msg": "Hello, World!" }`

---

## How to run the server

- Install the module with: \
  `pipenv install`
- Enter the environment: \
  `pipenv shell`
- Run the server with: \
  `flask run / python app.py`
